# Checklist

- Does the README render correctly on GitHub rather than only in local preview tools?
- Are forced line breaks written explicitly where visual separation matters?
- Are blockquotes separated correctly when multiple paragraphs are intended?
- Are image sizing rules compatible with GitHub sanitization?
- Do badge links and section links point to real anchors?
- Are external repository URLs canonical and up to date?
- Are obvious typos, naming issues, and numbering gaps corrected?
- Does the wording read like project documentation rather than AI-generated dialogue?
- Was the overall structure preserved unless a stronger change was clearly necessary?
